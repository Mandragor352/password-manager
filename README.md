# Password Manager

Простой консольный менеджер паролей на **C++** с локальным защищённым хранилищем.

Проект позволяет хранить, редактировать и удалять учётные данные, сохраняя их в зашифрованном виде в локальном файле.

## Возможности

* Добавление записей
* Просмотр всех записей
* Редактирование существующих записей
* Удаление записей
* Шифрование данных перед сохранением
* Защита мастер-паролем
* Хранение данных в локальном файле `vault.dat`

## Используемые технологии

* **C++17**
* **OpenSSL** — AES-256-GCM шифрование
* **Argon2** — вывод криптографического ключа из мастер-пароля
* **nlohmann/json** — сериализация данных

## Структура проекта

```text
password_manager/
│
├── main.cpp         # Точка входа
├── cmd.cpp/hpp      # Команды CLI
├── db.cpp/hpp       # Загрузка и сохранение vault
├── crypto.cpp/hpp   # Шифрование и дешифрование
├── data.hpp         # Структуры данных
├── vault.dat        # Зашифрованное хранилище
```

## Сборка

### Linux / macOS

```bash
g++ main.cpp cmd.cpp db.cpp crypto.cpp -o password_manager \
  -lssl -lcrypto -largon2
```

### Требования

Установленные библиотеки:

* OpenSSL
* Argon2
* nlohmann/json

Пример для Ubuntu:

```bash
sudo apt install libssl-dev libargon2-dev nlohmann-json3-dev
```


## Как работает защита данных

1. Пользователь вводит мастер-пароль
2. Из него создаётся ключ через **Argon2**
3. Данные шифруются с помощью **AES-256-GCM**
4. В файл сохраняются:

* salt
* nonce
* authentication tag
* ciphertext

При неверном пароле расшифровка невозможна.

## Ограничения текущей версии

* Ввод строк без пробелов
* Пароли отображаются открыто
* Нет генератора случайных паролей
* Нет поиска по записям
* Нет подтверждения при удалении
